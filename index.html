PCB Projects

PL Driver Evaluation Board

Function: A high-power, versatile driver evaluation board based on the TI TMS320F280049C. This board is designed for precise motor control applications with a high-efficiency power stage and advanced control algorithms by extending the full power of TMS320F280049C.

  • Microcontroller: TI TMS320F280049C based architecture
  • Power Ratings: 9V - 50V Input Voltage, Nominal 300W Output Power
  • Hardware Features: USB-C PD (@ 100W), eFuse protection, and Internal Magnetic Encoder
  • Communication & Interface: CANBus, UART, SPI, I2C, and JTAG programmable
  • Programming: Isolated USB Programmer for safe development

Autonomous System Status Indicator Board

Function: An ESP32-based status indicator board strictly designed to comply with the Formula Student Germany 2025 Rulebook, tailored for autonomous vehicle signaling and control.

  • Microcontroller: ESP32 based MCU architecture
  • Power Specifications: 12V Input Voltage with dedicated 1x 24V and 1x 12V power outputs
  • Compliance: Fully compliant with Formula Student Germany 2025 regulations
  • Communication & I/O: 1x CAN, 1x SPI, and 7x GPIO outputs
  • Indication & Interface: Capable of driving common cathode RGB LED strips, featuring a built-in encoder for a 7-position switch
  • Programming: Programmable via external USB to TTL converter

Brake System Plausibility Device (BSPD)

Function: A critical, strictly hardware-based safety system that continuously monitors brake and throttle inputs. It detects implausible states (e.g., simultaneous hard braking and acceleration) and autonomously opens the Shutdown Circuit (SDC) to ensure utmost driver safety.

  • Architecture: Standalone, non-programmable logic circuit designed for maximum reliability without any software dependencies.
  • Sensor Interfaces: Precision analog front-end supporting 1x Hall Effect Sensor (0-5V range) and 1x industrial-standard Pressure Sensor (4-20mA current loop).
  • Compliance: Engineered in strict compliance with the Formula Student Germany (FSG) 2025 Rulebook.
  • Safety Mechanism: Hardware-level implausibility detection that triggers an immediate SDC interruption to cut tractive system power.

NTC Slave PCB (Battery Temp Node)

Function: A compact Battery Management System (BMS) slave node designed to precisely monitor battery cell temperatures and relay critical thermal telemetry data to the main vehicle network.

  • Microcontroller: ESP32-based architecture for efficient data processing and network communication.
  • Thermal Sensing: 12-channel NTC thermistor input for comprehensive temperature monitoring of accumulator segments.
  • Communication: 1x CAN bus interface for reliable, real-time data transmission to the master BMS.
  • Safety & Reliability: Integrated short-circuit protection to ensure safe operation within the battery pack environment.
  • Programming: Accessible debug and programming interface via an external USB-to-TTL converter.
