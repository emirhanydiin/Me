## 🗂️ Featured Projects

The portfolio currently highlights projects across two main disciplines:

### 1. Hardware (PCB) Projects
Detailed features, components, and compliance standards (e.g., Formula Student Germany Rulebook) of my PCB designs:
* **PL Driver Evaluation Board:** A 300W motor control board based on TI TMS320F280049C.
* **Autonomous System Status Indicator:** An ESP32-based compliance and signaling board.
* **Brake System Plausibility Device (BSPD):** A standalone, purely hardware-based safety circuit.
* **NTC Slave PCB (Battery Temp Node):** An ESP32 BMS slave node for comprehensive thermal telemetry.

### 2. Embedded Software Projects 
*Descriptions of my algorithms and low-level programming architectures are available on the site. Direct links to the source code repositories will be added soon!*
* **Field-Oriented Control (FOC):** Bare-metal C implementation for motor control.
* **Custom Instruction Pipeline:** Computer architecture design and Verilog implementations.
